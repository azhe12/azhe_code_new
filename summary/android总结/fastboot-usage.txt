Fastboot Usage:

warning:when "waiting for device!"
	sudo fastboot flash ......

In fastboot mode
~~~~~~~~~~~~~~~~

Fastboot flash image command
    fastboot flash <partition> <filename>

partition:
  zip		- all image in the zip package
  boot		- boot.img
  zimage	- zImage
  hboot		- hboot.nb0
  system	- system.img
# userdata	- user data image
  dzdata	- dzdata.img
  rootfs	- root filesystem image

In hboot mode
~~~~~~~~~~~~~

Show log message on console:
	hboot> writeconfig 0 a
	hboot> writeconfig 9 1  (can disable off-mode)

Kernel will dump log messages to 'data/sd_err_msg':

	hboot> writeconfig 3 40

Print SD debug log on console:
	hboot> writeconfig 3 8

	hboot> writesku 1 0303FFFF
	hboot> writesku b E
	hboot> changeCid 11111111

Boot from SD card
~~~~~~~~~~~~~~~~~

1. Format SD card
2. Extract 'xloader' and 'hboot.nb0' from one dashboard zip package
3. Copy these two files to SD card, located at root directory
4. Rename them:
	xloader     ->   mlo
	hboot.nb0   ->   u-boot.bin
5. Reboot to hboot, that would boot from SD card
